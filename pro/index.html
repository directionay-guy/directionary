How to Play Directionary™

Goal

Complete 3 word puzzles using directional clues. Each guess shows whether letters should come earlier or later in the alphabet. Play as many times as you want!

Symbols Guide

● Correct letter in the right position
◄ Go earlier in the alphabet (your letter is too late)
► Go later in the alphabet (your letter is too early)

Example

If the target word is HOUSE and you guess GREAT:

G ► (H comes after G in alphabet)
R ◄ (O comes before R in alphabet)
E ► (U comes after E in alphabet)
A ► (S comes after A in alphabet)
T ◄ (E comes before T in alphabet)

AlphaHint™ - Interactive Hints (PRO Mode Only)

Stuck on a position? Hold any arrow in your most recent guess to see which letters are still possible.

How it works:
• Hold (press and keep pressing) an arrow ◄ or ► in your latest guess
• Invalid letters fade out, showing only valid possibilities
• The boundary arrows in previous guesses light up too

Example:
Your guesses for position 1:
• B► (target is after B in alphabet)
• D◄ (target is before D in alphabet)

Hold the D◄ → Only C remains visible
(The only letter after B but before D!)

Notes:
• Only works on your most recent guess (top row)
• Green dots ● are already solved - no hint needed
• Helps you understand the logic puzzle aspect of the game
• ⚠️ PRO+ Mode: AlphaHint is disabled for the harder challenge!

PRO Mode

Unlimited 3-round word puzzles with helpful features:

Features:
✓ AlphaHint enabled - Hold arrows to see valid letters
✓ Bold letters - Used letters are highlighted
✓ Can skip rounds - Get stuck? Skip and try the next word
✓ Any valid guess accepted - No restrictions on guessing

Scoring: Start with 100 points per word, lose 10 per incorrect guess.

PRO+ Mode (Hard Mode)

⭐ DOUBLE POINTS ⭐ for a hardcore challenge:

What's Different:
✗ No AlphaHint - Can't hold arrows to see valid letters
✗ No bold letters - Used letters don't highlight
✗ No skipping rounds - Must solve or fail each word
✗ Arrow constraints enforced - Invalid guesses rejected

Enforced Rules:
• If position shows ► - Next guess MUST use later letter
• If position shows ◄ - Next guess MUST use earlier letter
• If position shows ● - Next guess MUST use same letter
• Can't test with words you know are wrong!

Scoring: All scores ×2! Higher reward for the harder challenge.

Note: Pro and Pro+ have separate play counters and use different word sequences. You can play both modes!

Game Controls

Skip This Round:
• Skips the current round (PRO mode only)
• Scores that round as ZERO points
• You can still complete the remaining rounds
• ⚠️ PRO+ Mode: Skipping is disabled - you must solve each word!

Abandon Game:
• Small link below game buttons
• Ends current game immediately (all 3 rounds)
• Does NOT count toward stats or play count
• Starts fresh game with new words
• Use when truly stuck (especially in Pro+ mode)

Sharing

• The 3 lines of icons in the share screen represent the final guesses before the success or failure of each round.

Tips

• Valid words include common English words, but exclude proper nouns, names, profanity, and specialized jargon. Slang may be appropriate in some instances. No plural or past tense words are the answers but may be used as guesses.
• The alphabet guide shows which letters you've tried per guess - darkened letters show what you've used, but they may still be in the word at different positions.
• Start with words using middle alphabet letters (L-M-N-O) to eliminate half the alphabet at once.
• Pay attention to the arrows - they show which direction to move through the alphabet.
